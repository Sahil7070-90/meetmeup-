# 🎥 MeetMeUp

Random video chat website — Talk to strangers instantly!

## Features
- 🎥 Live Video & Audio Chat
- 💬 Real-time Text Chat
- ⚡ Instant Random Matching
- 🔒 100% Anonymous
- 👥 Online User Counter

## Tech Stack
- Node.js + Express
- Socket.io (Real-time matching)
- WebRTC + PeerJS (Video chat)
- Vanilla HTML/CSS/JS
